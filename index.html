Colors Cheat Sheet

Name keywords

Color name keywords can be used to set color property values for elements in CSS.

.color-sec code {
  background-color: midnightblue;
  color: lavenderblush;
}

Rgb colors

CSS colors can be declared with RGB colors using rgb() syntax. rgb() should be supplied with three values representing red, green, and blue. These values range can from 0 to 255.

.color-sec code {
  background-color: rgb(25, 25, 12);
  color: rgb(255, 240, 245);
}

HSL scale

CSS colors can be declared with the HSL color system using hsl() syntax. This syntax contains three values: hue (the color value itself), saturation (intensity), and lightness. Hue values range from 0 to 360 while saturation and lightness values are represented as percentages.

.color-sec code {
  background-color: hsl(240, 64%, 27%);
  color: hsl(340, 100%, 97%);
}

Hexadecimals

CSS colors can be represented in hexadecimal (or hex) notation. Hexadecimal digits can represent sixteen different values using 0-9 and a-f. When both characters of all three colors are repeated, hex colors can be abbreviated to only three values, so #0000ff could also be represented as #00f.

.color-sec code {
  background-color: #191970;
  color: #fff0f5;
}
